TA-095 PRE-Downgrader fix - 2012, frostegater

RU:
-----------------

- Использование:
1. Установите официальную прошивку 6.60.
2. Установите кастомную прошивку 6.60 LME-1.6.
3. Сделайте NAND дамп с помощью программы NAND Tool.
4. Запустите 'PSP TA-095v2 pre-downgrader' и нажмите Х для выхода.
5. Запустите 'Downgrader v5b' и откатите прошивку на 6.20.

EN:
-----------------

1. Install 6.60 OFW.
2. Install CFW LME-1.6.
3. Make NAND dump by NAND Tool.
4. Run 'PSP TA-095v2 pre-downgrader' and press X to exit.
5. Run 'Downgrader v5b' and make downgrade.
